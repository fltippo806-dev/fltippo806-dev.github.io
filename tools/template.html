口径说明（v1.0 · 已拍板）

  • 消耗 = 渠道侧消耗 network_cost（FB/GG/TT 统一；TikTok 归因口径消耗恒为 0，故全渠道统一采用渠道侧）。
  • 注册 = signup_complete 事件；CPI = 消耗÷安装；注册成本 CPA = 消耗÷注册。
  • D0 付费率 = D0 付费用户 ÷ 注册。
  • DX ROAS = 批次 X 日累计收入 ÷ 对应消耗（未扣退款）。仅统计"窗口已走完"的成熟批次：D3 列用截至 T-3 的 28 天批次，D7 列用截至 T-7，D30 列用截至 T-30，以此类推——因此不同列的统计窗口不同，横向对比同列、不要跨列对比时间。
  • 自然量（Organic / Google Organic Search）不参与任何 ROI 计算；Kita 未投放，不纳入看板。
  • 颜色标尺：D0 ROAS ≥0.6 绿 / 0.3–0.6 黄 / <0.3 红；D7 ≥1.0 绿 / 0.7–1.0 黄 / <0.7 红；D30 ≥1.3 绿 / 1.0–1.3 黄 / <1.0 红。为经验参考值，正式基准表产出后替换。
  • Campaign 信号（参考）：🔴 关注止损 = 近14天消耗≥$100 且 D0 ROAS<0.30；🟢 可放量 = 消耗≥$80 且 D0 ROAS≥0.60。正式止损/放量线待基准建模后生效。
  • 数据 T+1；每天早上 09:00（新加坡时间）自动更新。
